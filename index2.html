Ejercicio 1: Listas.

Lautaro Morales 4°to 1°ra.

1.1- ¿Para qué sirve la etiqueta ul? Dar un ejemplo en donde lo utilizarías.

La etiqueta "ul" (unordered list)-(lista no ordenada). crea una lista la cual no es ordenada. Ej: Para plantear cosas.

  • Auto
  • Camion
  • Moto

1.2- ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías.

La etiqueta "ol" permite definir listas ordenadas (Ordered List), bien con numeración o alfabéticamente. Ej: Una playlist de música.

  1. Te para tres
  2. Giros
  3. La pregunta

1.3- ¿Para qué sirve la etiqueta dl? Dar un ejemplo en donde lo utilizarías.

La etiqueta "dl" permite hacer listas descriptivas. El elemento encierra una lista de grupos de términos y de descripciones. Ej: Describir una pagina.

YouTube
YouTube Music
YTM
Es una App para escuchar musica libremente con los benificios que te trae YouTube.

1.4- ¿Para qué sirve la etiqueta li? Dar un ejemplo en donde lo utilizarías.

"OL" y "LI" sirven para crear listas, OL para listas ordenadas y los elementos se identifican con la etiqueta LI (no puede ir solita). Ej:

  • Punto uno
  • Punto dos
  • Punto tres

Ejercicio 2: Enlaces.

Lautaro Morales 4°to 1°ra.

2.1- ¿qué es una url absoluta?

Una URL absoluta contiene toda la información necesaria para localizar un recurso.

2.2- ¿qué es una url relativa?

Una URL relativa localiza un recurso mediante una dirección URL absoluta como punto de partida.
La "dirección URL completa" del destino se especifica concatenando las direcciones URL absoluta y relativa.
